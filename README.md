# Cricket Match Scheduler

Simple cricket match scheduler for local teams.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start server:
   ```bash
   npm start
   ```

3. Open browser: `http://localhost:3000`

## How It Works

1. Sign up as Captain
2. Create team
3. Post availability (day + bet amount)
4. System auto-matches teams
5. Both captains confirm match

## Features

- User registration/login
- Team creation
- Availability posting
- Automatic matching
- Match confirmation
- Admin panel
- Chat between captains

## Tech Stack

- Node.js + Express
- Vanilla HTML/CSS/JS
- JSON file storage
- JWT authentication
